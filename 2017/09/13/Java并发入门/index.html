Java并发入门

发表于 2017-09-13 | 分类于 Java并发 |

网上看了一篇讲Java线程较为基础但是讲的很详细的博客，很多东西以前是学过的，但是用的比较少就忘了，这里还是好好学下。

参考资料

  • Java线程(二)-线程同步synchronized和volatile: http://blog.csdn.net/ghsau/article/details/7424694

  • Java并发编程-volatile关键字解析:http://www.importnew.com/18126.html

(二)这篇文章对volatile关键字得解析有误，结合并发编程volatile关键字解析看会理解更深刻

  • Java线程(三)-线程协作-生产者/消费者问题:http://blog.csdn.net/ghsau/article/details/7433673

  • Java线程(四)-线程中断、线程让步、线程睡眠、线程合并:http://blog.csdn.net/ghsau/article/details/17560467

  • Java线程中yield与join方法的区别-http://www.importnew.com/14958.html

(四)这篇文章作者对yield理解可能是对的，但是文章中说了不清不楚，有人提醒了还不改，建议结合yield和join方法区别结合来看

  • Java线程(五)-Timer和TimerTask：http://blog.csdn.net/ghsau/article/details/7436458

  • 详解java定时任务-http://www.importnew.com/19573.html

(五)这篇文章对timer的讲解不足，结合详解java定时任务看会收获多一点

  • Java线程(六)-线程池：http://blog.csdn.net/ghsau/article/details/7443324
  • Java线程(七)-Callable和Future:http://blog.csdn.net/ghsau/article/details/7451464
  • Java线程(八)-锁对象Lock-同步问题更完美的处理方式:http://blog.csdn.net/ghsau/article/details/7461369
  • Java线程(九)-Condition-线程通信更高效的方式：http://blog.csdn.net/ghsau/article/details/7481142
  • Java线程(十)-CAS：http://blog.csdn.net/ghsau/article/details/38471987

这一章讲的概念AtomicInteger保证原子性，CAS：compare and set

  • Java线程(十一)-Fork/Join-Java并行计算框架：http://blog.csdn.net/ghsau/article/details/46287769
  • Java线程(篇外篇)-阻塞队列BlockingQueue：http://blog.csdn.net/ghsau/article/details/8108292
