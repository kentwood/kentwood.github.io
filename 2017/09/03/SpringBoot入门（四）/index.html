事务管理

事务管理的基本概念这里就不多说了，参考博客Spring框架（四）。简单的说就是两条 sql 语句同时在一个方法中执行，为了防止一个 sql 语句执行成功而另一个 sql 语句执行失败，引入了事务管理。

SpringBoot为方法添加事务的方式也十分简单。只需要在方法上添加一个@Transaction就可以。试验一下，先添加一个service包，在service包里添加FattyService类
